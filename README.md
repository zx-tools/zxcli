# zxcli
ZX Command Line Interface

## Installation

```bash
npm install -g zxcli
```

## Usage

```bash
zx hello
```

## Development

### Building

```bash
npm run build
```

### Testing

Run the test suite:

```bash
npm test
```

Run tests in watch mode:

```bash
npm run test:watch
```

Run tests with coverage:

```bash
npm run test:coverage
```

### Development mode

Run the CLI in development mode with live reloading:

```bash
npm run dev hello
```
